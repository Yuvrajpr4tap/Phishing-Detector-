About This Detector

This phishing email detector uses Natural Language Processing (NLP) and Machine Learning to identify suspicious emails.

How it works:

  • TF-IDF Vectorization: Converts email text into numerical features
  • Machine Learning: Uses Naive Bayes or Logistic Regression models trained on email samples
  • Confidence Score: Shows how confident the model is in its prediction

Tech Stack:

  • Python
  • Scikit-learn (ML library)
  • Pandas (Data processing)
  • Flask (Web framework)

⚠️ Disclaimer:

This tool is for educational purposes. Always verify emails independently and use additional security measures for important accounts.
